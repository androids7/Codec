# Codec
Android encoding and decoding library
用C实现的安卓解码与编码库。

目前支持的算法：

- BASE64
- MD5

##引入库

for Gradle:
```
compile 'me.majiajie:codec:1.0.0'
```

and for Maven:
```
<dependency>
  <groupId>me.majiajie</groupId>
  <artifactId>codec</artifactId>
  <version>1.0.0</version>
  <type>pom</type>
</dependency>
```
